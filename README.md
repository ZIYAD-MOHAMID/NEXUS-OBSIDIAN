# ◈ NEXUS OBSIDIAN

> **The Neural Architecture for Full-Stack Mastery.**

Nexus Obsidian is a high-performance, minimalist cognitive manager designed for developers. It transforms fluid information into permanent expertise using a "Neural Pathway" pipeline, leveraging active recall and structured memory anchoring.

---

## ⚡ Core Philosophy

In a world of information overload, facts tend to drift. Nexus Obsidian provides the **Anchors**. By categorizing knowledge into evolutionary stages, it ensures that what you learn today becomes part of your professional architecture tomorrow.

### The Neural Pathways

- **Genesis:** The entry point for raw data. Capture new syntax and logic.
- **Study:** Active recall protocol. Strengthen neural links through retrieval cycles.
- **Vault:** Your Source of Truth. A permanent library of hard-coded expertise.
- **QNA:** Rapid-fire modules optimized for technical interview readiness.

---

## 🛠️ Tech Stack

Built with a focus on speed, persistence, and a "Mobile-First" terminal aesthetic.

- **Frontend:** Vanilla JavaScript, HTML5, CSS3 (Custom Glassmorphism & Bento Grid UI)
- **State Management:** LocalStorage Persistence
- **Design:** CSS Variables, Flexbox/Grid Architecture, Radial Gradient Engine
